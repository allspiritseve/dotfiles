# Dotfiles

## New Machine

1. Install [iTerm](https://iterm2.com)

## Instructions

1. Clone repo

    git clone git@github.com:allspiritseve/dotfiles.git ~/machine

1. Install Homebrew

    /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

1. Run `make`

    cd ~/machine
    make
